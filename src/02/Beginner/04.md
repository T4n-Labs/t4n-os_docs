# Panduan Instalasi

Setelah kamu:

- Mengunduh ISO [T4n OS](../[01.md]#file-iso)
- Membuat media bootable (USB/DVD)

Kamu sudah siap untuk menginstal sistem.

## ⚠️ Persiapan Penting

Sebelum mulai, pastikan kamu mengetahui jenis firmware sistem kamu:

- **BIOS (Legacy)**, atau
- **UEFI (Modern)**

Ini akan memengaruhi skema partisi yang digunakan. Lihat [dokumentasi partisi](03.md) untuk referensi.

**Fitur yang tidak didukung installer:**
- LVM
- LUKS (enkripsi disk)
- ZFS

## Booting

1. Nyalakan komputer
2. Masuk ke boot menu — biasanya `F2`, `F12`, `ESC`, atau `DEL`
3. Pilih USB / DVD T4n OS

> **Tips:** Tersedia opsi **load ke RAM**. Proses awal lebih lambat, tetapi instalasi berjalan lebih cepat setelahnya.


## Masuk ke Live System

Login dengan kredensial berikut:

| Field    | Value              |
|----------|--------------------|
| Username | `root` atau `anon` |
| Password | `voidlinux`        |

> NOTE : username `root` untuk Root dan username `anon` untuk User

## Installer

T4n OS menyediakan dua pilihan installer:

### 1. CLI Installer
Installer berbasis menu (TUI).

```bash
t4n-installer
```

### 2. GUI Installer (ALPHA Version)

```bash
t4n-installer-gui
```

> Dibangun dengan Python. Langkah konfigurasi sama seperti CLI installer di bawah ini.

## Konfigurasi Installer

### Keyboard

Pilih layout keyboard yang sesuai. Gunakan `us` untuk keyboard standar.


### Jaringan

T4n OS menggunakan **NetworkManager** sebagai pengelola jaringan default.

##### Kabel (Ethernet)

Koneksi biasanya terdeteksi otomatis melalui DHCP.

##### WiFi

isi informasi berikut:

- **SSID** — nama jaringan WiFi
- **Enkripsi** — `wpa` atau `wep`
- **Password**

> **Jika koneksi gagal:**
> Keluar dari installer, lalu konfigurasi jaringan secara manual menggunakan `wpa_supplicant` dan `dhcpcd`.

### Source (Sumber Instalasi)

Pilih sumber paket instalasi:

| Opsi      | Keterangan                         |
|-----------|------------------------------------|
| `Local`   | Instalasi dari ISO (offline)       |
| `Network` | Mengunduh paket terbaru dari repo  |

> ⚠️ **Jika menggunakan ISO XFCE, wajib pilih `Local`.**

### Hostname

Masukkan nama komputer. Contoh:

```
T4nLabs
```

### Locale

Pilih bahasa dan pengaturan regional. Contoh:

- `en_US.UTF-8`
- `id_ID.UTF-8`

### Timezone

Pilih zona waktu. Contoh:

- `Asia/Jakarta`

### Password Root

Masukkan password untuk akun `root`.

> Karakter tidak akan terlihat saat mengetik — ini normal.

### User

Buat akun pengguna baru:

- **Username** (default: `void`)
- **Nama lengkap**
- **Password**

> User yang dibuat otomatis masuk ke grup `wheel`, sehingga dapat menggunakan `sudo`.

### Partisi
Pilih tool partisi yang tersedia sesuai installer yang digunakan:

| Tool    | CLI Installer | GUI Installer |
|---------|---------------|---------------|
| cfdisk  |       ✅      |       ✅      |
| fdisk   |       ✅      |       ✅      |
| gparted |       ❌      |       ✅      |



**Langkah:**

1. Pilih disk target (contoh: `/dev/sda`)
2. Buat partisi yang diperlukan:
   - Root `/`
   - Swap
   - `/boot/efi` *(khusus UEFI)*
3. Pilih **Write**, lalu **Quit** (atau Apply untuk GParted)

| Firmware | Skema Partisi       |
|----------|---------------------|
| UEFI     | GPT + EFI partition |
| BIOS     | MBR                 |

### Filesystem

Untuk setiap partisi, tentukan:

- **Filesystem:**
  - `ext4` *(disarankan untuk `/` dan `/home`)*
  - `vfat` *(untuk partisi EFI)*
- **Mount point:**
  - `/`
  - `/boot/efi`
  - `/home` *(opsional)*

### Bootloader

Pilih lokasi instalasi bootloader:

- Pilih disk (contoh: `/dev/sda`), atau
- Pilih `none` untuk konfigurasi manual

Bootloader yang digunakan: **GRUB**

### Review

Periksa semua konfigurasi sebelum melanjutkan. Pastikan:

- Disk yang dipilih sudah benar
- Mount point sesuai
- Bootloader sudah dikonfigurasi

## Proses Instalasi

Pilih **Install** untuk memulai. Installer akan secara otomatis:

1. Memformat partisi
2. Menginstal sistem dasar
3. Membuat `initramfs`
4. Menginstal GRUB

⏳ Tunggu hingga proses selesai.

## Selesai & Reboot

Setelah instalasi sukses jalankan:

```bash
reboot
```

Lalu cabut media USB/DVD dari Device

## Pasca Instalasi

Setelah masuk ke sistem, segera lakukan pembaruan:

```bash
sudo xbps-install -Su
```

Lihat [Dokumentasi XBPS](#) untuk penggunaan lebih lanjut.
