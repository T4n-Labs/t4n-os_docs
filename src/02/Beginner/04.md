# Panduan Instalasi
Setelah kamu:
* Mengunduh ISO T4n OS
* Membuat media bootable (USB/DVD)

Sekarang kamu siap untuk menginstal sistem.

## ⚠️ Persiapan Penting

Sebelum mulai:

* Pastikan kamu tahu sistem kamu:
  * **BIOS (Legacy)** atau
  * **UEFI (Modern)**
* Ini akan memengaruhi partisi ([lihat dokumentasi partisi sebelumnya](./02/Beginner/03.md)).

📌 Fitur yang **tidak didukung installer**:

* LVM
* LUKS (enkripsi disk)
* ZFS

## Booting

1. Nyalakan komputer
2. Masuk ke boot menu (biasanya `F2`, `F12`, `ESC`, atau `DEL`)
3. Pilih USB / DVD T4n OS

📌 Tips:

* Ada opsi **load ke RAM**
* Lebih lambat saat awal, tapi instalasi jadi lebih cepat

## Masuk ke Live System

Login sebagai:

* Username: `root`
* Password: `voidlinux`

Lalu jalankan installer:

```bash
# void-installer
```

## Konfigurasi Installer
Installer berbasis menu (TUI). Ikuti langkah berikut:

### Keyboard
Pilih layout keyboard.
📌 Umumnya:

* Gunakan `us` untuk keyboard standar

### Jaringan
Pilih koneksi:
#### 🔹 Kabel (Ethernet)
* Biasanya otomatis (DHCP)

#### 🔹 WiFi
Isi:
* SSID (nama WiFi)
* Enkripsi (`wpa` / `wep`)
* Password

📌 Jika gagal konek:

* Keluar installer
* Gunakan:

  * `wpa_supplicant`
  * `dhcpcd`

### Source (Sumber Instalasi)
Pilih sumber paket:

* `Local` → dari ISO
* `Network` → download terbaru

⚠️ Penting:
> Jika pakai ISO XFCE, WAJIB pilih **Local**

### Hostname

Masukkan nama komputer:
Contoh:
```
T4nLabs
```

### Locale
Pilih bahasa & regional.
Contoh:

* `en_US.UTF-8`
* `id_ID.UTF-8`

### Timezone
Pilih zona waktu:

Contoh:
* `Asia/Jakarta`


### Password Root
Masukkan password untuk root.

📌 Tidak akan terlihat saat mengetik.

### User
Buat user baru:
* Username (default: `void`)
* Nama lengkap
* Password

📌 User otomatis masuk ke grup:

* `wheel` → bisa pakai `sudo`

### Bootloader
Pilih lokasi install bootloader:

* Pilih disk (contoh: `/dev/sda`)
* Atau pilih `none` (manual)

Bootloader yang digunakan:
👉 GRUB


### Partisi
Installer akan membuka:

```bash
cfdisk
```

Langkah:

1. Pilih disk
2. Buat partisi:

   * Root `/`
   * Swap
   * (UEFI) `/boot/efi`
3. Pilih **Write** lalu **Quit**

📌 Penting:

* UEFI → GPT + EFI partition
* BIOS → MBR

### Filesystem

Untuk setiap partisi:

* Pilih filesystem:

  * `ext4` (disarankan)
  * `vfat` untuk EFI
* Tentukan mount point:

  * `/`
  * `/boot/efi`
  * `/home` (opsional)

### Review
Periksa semua konfigurasi.
📌 Pastikan:

* Disk benar
* Mount point sesuai
* Bootloader sudah dipilih

## Proses Instalasi

Pilih:
```
Install
```

Installer akan otomatis:

* Format partisi
* Install sistem dasar
* Membuat initramfs
* Install GRUB

⏳ Tunggu sampai selesai.

## Selesai & Reboot

Jika sukses:

```bash
reboot
```

⚠️ Cabut USB saat reboot

## Pasca Instalasi

Setelah masuk ke sistem:`Update sistem`

```bash
sudo xbps-install -Su
```
[Dokumentasi XBPS](#)

## Kesimpulan

* Installer T4n OS berbasis menu (mudah digunakan)
* Partisi harus dibuat manual
* Gunakan:

  * GPT + EFI untuk UEFI
  * MBR untuk BIOS
* Setelah install → wajib update sistem
