# Firmware

T4n OS menyediakan sejumlah paket firmware di repositori. Beberapa firmware hanya tersedia jika repositori [nonfree](#) sudah diaktifkan.

## Microcode

Microcode dimuat ke CPU atau GPU saat booting oleh BIOS, namun dapat diperbarui kemudian oleh sistem operasi. Pembaruan microcode memungkinkan perilaku CPU atau GPU dimodifikasi untuk mengatasi bug tertentu — tanpa perlu mengganti hardware.

### Intel

Instal paket microcode Intel:

```bash
sudo xbps-install -S intel-ucode
```

> Paket ini berada di repo **nonfree** — pastikan sudah [diaktifkan](#) terlebih dahulu.

Setelah instalasi, regenerasi [initramfs](#) secara manual:

```bash
sudo xbps-reconfigure -f linux<versi>
```

Untuk pembaruan selanjutnya, microcode akan ditambahkan ke initramfs secara otomatis.

### AMD

Instal paket firmware AMD:

```bash
sudo xbps-install -S linux-firmware-amd
```

Paket ini mencakup microcode untuk CPU maupun GPU AMD. Microcode akan dimuat secara otomatis — tidak ada konfigurasi tambahan yang diperlukan.


## Verifikasi

Untuk memverifikasi bahwa microcode berhasil dimuat, periksa informasi di `/proc/cpuinfo`:

```bash
grep microcode /proc/cpuinfo
```

## Menghapus Firmware

Secara default, paket `linuxX.Y` dan `base-system` menginstal sejumlah paket firmware. Firmware yang tidak terpakai tidak wajib dihapus, namun jika ingin melakukannya:

1. Konfigurasi XBPS untuk [mengabaikan](#) paket firmware yang tidak diinginkan
2. Hapus paket tersebut menggunakan:

```bash
sudo xbps-remove -R <nama-paket-firmware>
```
