# Dokumentasi T4n OS Base

**T4n OS Base** adalah varian instalasi minimal (CLI-only) yang dirancang untuk pengguna yang menginginkan kontrol penuh atas sistem mereka. Versi ini tidak menyertakan antarmuka grafis (GUI) secara bawaan, memberikan fondasi yang ringan, cepat, dan stabil.

## 1. Filosofi Sistem
T4n OS Base dibangun dengan prinsip:
*   **Minimalisme:** Hanya paket esensial yang terinstal.
*   **Kontrol Pengguna:** Anda memutuskan apa yang akan ditambahkan ke sistem.
*   **Stabilitas:** Menggunakan kernel LTS dan sistem inisialisasi yang efisien.

---

## 2. Manajemen Paket (`xbps`)
T4n OS menggunakan sistem paket **XBPS** (X Binary Package System). Berikut adalah perintah dasar yang perlu Anda ketahui:

*   **Sinkronisasi Repositori & Update Sistem:**
```bash
    sudo xbps-install -Su
```
*   **Mencari Paket:**
    
```bash
    xbps-query -Rs <nama_paket>
```
*   **Menginstal Paket Baru:**
    
```bash
    sudo xbps-install -S <nama_paket>
```
*   **Menghapus Paket:**
```bash
    sudo xbps-remove -R <nama_paket>
```
*   **Membersihkan Cache Paket:**
```bash
    sudo xbps-remove -O
```

---

## 3. Manajemen Layanan (`runit`)
T4n OS menggunakan **runit** sebagai sistem *init* dan manajemen layanan karena kecepatannya.

*   **Melihat Layanan yang Sedang Berjalan:**
    ```bash
    ls /var/service/
    ```
*   **Mengaktifkan Layanan:**
    Cukup buat *symbolic link* dari `/etc/sv/` ke `/var/service/`:
    
```bash
    sudo ln -s /etc/sv/<nama_layanan> /var/service/
```
*   **Menghentikan/Memulai Layanan:**
    
```bash
    sudo sv up <nama_layanan>    # Memulai
    sudo sv down <nama_layanan>  # Menghentikan
    sudo sv restart <nama_layanan> # Muat ulang
```
*   **Mengecek Status Layanan:**
    
```bash
    sudo sv status <nama_layanan>
```

---

## 4. Konfigurasi Jaringan
Pada versi Base, Anda dapat mengelola jaringan menggunakan `dhcpcd` untuk koneksi kabel atau `NetworkManager` jika sudah diinstal.

*   **Mengaktifkan Internet (Kabel):**
    
```bash
    sudo ln -s /etc/sv/dhcpcd /var/service/
```
*   **Mengatur Nama Host (Hostname):**
    Edit file `/etc/hostname`:
    ```bash
    echo "t4n-laptop" | sudo tee /etc/hostname
    ```

---

## 5. Manajemen Pengguna
Secara default, instalasi base mungkin hanya memiliki akses `root` atau satu user admin.

*   **Menambah User Baru:**
    
```bash
    sudo useradd -m -G wheel,users,video,audio -s /bin/bash <username>
```
*   **Mengatur Password:**
```bash
    sudo passwd <username>
```
*   **Memberikan Akses Sudo:**
    Pastikan paket `sudo` terinstal, lalu hapus tanda komentar pada baris `%wheel ALL=(ALL) ALL` di file `/etc/sudoers` menggunakan perintah `visudo`.

---

## 6. Langkah Selanjutnya (Opsional)
Setelah sistem Base siap, Anda bisa melanjutkan ke instalasi komponen spesifik:
*   **Xorg:** Untuk dukungan display server.
*   **Wayland:** Untuk protokol grafis modern.
*   **GPU Drivers:** Instal `linux-firmware-intel`, `nvidia`, atau driver AMD sesuai perangkat Anda.
*   **Desktop Environment:** Instal `xfce4`, `gnome`, atau `plasma-desktop`.

---

## 7. Lokasi File Penting
| Jalur File | Deskripsi |
| :--- | :--- |
| `/etc/xbps.d/` | Konfigurasi repositori tambahan. |
| `/etc/runit/` | Skrip inisialisasi sistem. |
| `/var/log/` | Lokasi log sistem. |
| `/etc/fstab/` | Konfigurasi mounting partisi disk. |
