# Multimedia: ALSA (Advanced Linux Sound Architecture)

**ALSA** menyediakan fungsionalitas audio dasar pada tingkat kernel. Dokumentasi ini menjelaskan cara mengonfigurasi ALSA sebagai backend audio utama pada T4n OS.

## 1. Instalasi dan Persiapan

1.  **Instalasi Paket:**
    Instal paket utilitas dasar ALSA:
    ```bash
    sudo xbps-install -S alsa-utils
    ```

2.  **Izin Pengguna:**
    Pastikan pengguna Anda terdaftar dalam grup `audio` untuk mendapatkan akses ke perangkat keras:
    ```bash
    sudo usermod -aG audio $USER
    ```

3.  **Mengaktifkan Layanan:**
    Paket `alsa-utils` menyediakan layanan `alsa` yang berfungsi menyimpan dan memulihkan status volume saat sistem dimatikan atau dinyalakan. Aktifkan layanan ini:
    ```bash
    sudo ln -s /etc/sv/alsa /var/service/
    ```

---

## 2. Kompatibilitas PulseAudio via `apulse`

Beberapa aplikasi mewajibkan adanya PulseAudio. Jika Anda hanya ingin menggunakan ALSA, Anda dapat menggunakan `apulse` sebagai lapisan kompatibilitas yang menerjemahkan instruksi PulseAudio langsung ke ALSA.

*   **Instalasi:** `sudo xbps-install -S apulse`
*   **Penggunaan:** Jalankan aplikasi dengan awalan `apulse`.
    *Contoh:* `apulse firefox`

---

## 3. Konfigurasi Kartu Suara Default

Anda dapat menentukan kartu suara default melalui file konfigurasi ALSA atau opsi modul kernel.

### **Identifikasi Kartu Suara**
Periksa urutan modul kartu suara yang dimuat oleh sistem:
```bash
cat /proc/asound/modules
```
*Contoh output:*
` 0 snd_hda_intel`
` 1 snd_hda_intel`
` 2 snd_usb_audio`

### **Metode A: Melalui Konfigurasi ALSA**
Edit file `/etc/asound.conf` (global) atau `~/.asoundrc` (per-user). Untuk menjadikan kartu nomor 2 sebagai default:
```text
defaults.ctl.card 2;
defaults.pcm.card 2;
```

### **Metode B: Melalui Modul Kernel**
Anda dapat memaksa urutan indeks kartu suara di `/etc/modprobe.d/alsa.conf`. Contoh agar audio USB selalu menjadi indeks pertama (0):
```text
options snd_usb_audio index=0
```

---

## 4. Pengaturan Multi-Sumber (Dmix)

Plugin `dmix` memungkinkan ALSA memutar suara dari beberapa aplikasi secara bersamaan (software mixing). Secara default, `dmix` aktif otomatis jika kartu suara tidak mendukung *hardware mixing*.

Jika Anda ingin mengaktifkannya secara eksplisit untuk output digital, tambahkan baris berikut ke `/etc/asound.conf`:

```text
pcm.dsp {
    type plug
    slave.pcm "dmix"
}
```

---

## 5. Tips Tambahan
*   **Mixer:** Gunakan perintah `alsamixer` di terminal untuk mengatur volume secara grafis melalui konsol. Tekan `F6` untuk memilih kartu suara yang berbeda.
*   **Penyimpanan Volume:** Jalankan `sudo alsactl store` jika Anda ingin menyimpan pengaturan volume saat ini secara manual.
