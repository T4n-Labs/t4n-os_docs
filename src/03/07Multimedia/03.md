# Multimedia: PipeWire

**PipeWire** adalah server modern untuk menangani aliran audio dan video. PipeWire sangat fleksibel dan dapat terhubung dengan aplikasi yang dirancang untuk sistem audio ALSA, PulseAudio, dan JACK. PipeWire juga mendukung aplikasi Flatpak serta menyediakan metode untuk berbagi layar (*screensharing*) di Wayland melalui `xdg-desktop-portal`.

## 1. Persiapan (Prerequisites)

Sebelum mengaktifkan PipeWire, pastikan sistem memenuhi syarat berikut:
*   **D-Bus User Session:** PipeWire memerlukan sesi D-Bus pengguna yang aktif. Jika lingkungan desktop atau *window manager* Anda belum menyediakannya, jalankan sesi dengan `dbus-run-session`.
*   **Variabel Lingkungan:** Variabel `XDG_RUNTIME_DIR` harus didefinisikan agar PipeWire berfungsi dengan benar.
*   **Grup Pengguna:** Jika tidak menggunakan `elogind`, masukkan user Anda ke grup `audio` (untuk suara) dan `video` (untuk kamera/layar):
    ```bash
    sudo usermod -aG audio,video $USER
    ```

---

## 2. Instalasi Dasar

Instal paket utama PipeWire. Ini juga akan menginstal **WirePlumber** sebagai manajer sesi bawaan:
```bash
sudo xbps-install -S pipewire
```

### **Konfigurasi Manajer Sesi (WirePlumber)**
PipeWire tidak akan berfungsi tanpa manajer sesi. Untuk memastikan PipeWire menjalankan WirePlumber secara otomatis, buat tautan konfigurasi berikut:

**Opsi A: Untuk Seluruh Sistem (Global)**
```bash
sudo mkdir -p /etc/pipewire/pipewire.conf.d
sudo ln -s /usr/share/examples/wireplumber/10-wireplumber.conf /etc/pipewire/pipewire.conf.d/
```

**Opsi B: Per Pengguna (User)**
```bash
mkdir -p ~/.config/pipewire/pipewire.conf.d
ln -s /usr/share/examples/wireplumber/10-wireplumber.conf ~/.config/pipewire/pipewire.conf.d/
```

---

## 3. Antarmuka PulseAudio (Sangat Direkomendasikan)

Banyak aplikasi tidak bisa berkomunikasi langsung dengan PipeWire dan membutuhkan antarmuka PulseAudio.
1.  Hapus paket `pulseaudio` jika masih terinstal.
2.  Aktifkan emulator PulseAudio di PipeWire:

**Opsi A: Global**
```bash
sudo ln -s /usr/share/examples/pipewire/20-pipewire-pulse.conf /etc/pipewire/pipewire.conf.d/
```

**Opsi B: User**
```bash
ln -s /usr/share/examples/pipewire/20-pipewire-pulse.conf ~/.config/pipewire/pipewire.conf.d/
```

---

## 4. Pengujian dan Menjalankan Otomatis

### **Pengujian Manual**
Jalankan perintah berikut di terminal untuk memastikan tidak ada error:
```bash
pipewire
```
Cek status WirePlumber dengan:
```bash
wpctl status
```
Jika antarmuka PulseAudio aktif, cek dengan:
```bash
pactl info
# Harus muncul: Server Name: PulseAudio (on PipeWire 0.x.x)
```

### **Menjalankan Otomatis (Autostart)**
Setelah dipastikan bekerja, Anda bisa menjalankannya otomatis saat login:
*   **Desktop Environment:** Tambahkan `pipewire` ke daftar *Startup Applications*.
*   **XDG Autostart:** Buat symlink dari file desktop ke folder autostart:
    
```bash
    ln -s /usr/share/applications/pipewire.desktop ~/.config/autostart/
```
*   **Script Window Manager:** Tambahkan baris `pipewire &` pada file `~/.xinitrc` atau script startup WM Anda.

## 6. Pemecahan Masalah (Troubleshooting)

### **Error Umum**
*   **"Failed to connect to system bus"**: Layanan D-Bus sistem tidak berjalan. Aktifkan dengan `sudo ln -s /etc/sv/dbus /var/service/`.
*   **"XDG_RUNTIME_DIR not set"**: Pastikan variabel lingkungan ini sudah diatur (biasanya oleh `elogind` atau manual di `.bash_profile`).
*   **Hanya muncul "Dummy Output"**: Biasanya karena WirePlumber belum berjalan atau user belum masuk grup `audio`.

### **Integrasi ALSA Gagal**
Jika aplikasi ALSA tidak mengeluarkan suara melalui PipeWire, pastikan konfigurasi berikut sudah dibuat:
```bash
sudo mkdir -p /etc/alsa/conf.d
sudo ln -s /usr/share/alsa/alsa.conf.d/50-pipewire.conf /etc/alsa/conf.d
sudo ln -s /usr/share/alsa/alsa.conf.d/99-pipewire-default.conf /etc/alsa/conf.d
```
