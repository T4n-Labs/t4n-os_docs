# Multimedia
## Pengaturan audio

Untuk mengatur audio pada sistem Void Linux Anda, Anda harus memutuskan apakah ingin menggunakan PulseAudio, PipeWire, atau hanya ALSA. Sndio juga tersedia, tetapi tidak didukung dan tidak direkomendasikan.

Beberapa aplikasi membutuhkan PulseAudio, terutama program sumber tertutup, tetapi PipeWire menyediakan pengganti langsung untuk PulseAudio.

Jika elogind tidak diaktifkan, Anda perlu berada dalam grup audio agar dapat mengakses perangkat audio.
