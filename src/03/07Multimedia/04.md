# Multimedia: PulseAudio

**PulseAudio** adalah server suara yang berfungsi sebagai perantara antara aplikasi dan perangkat keras audio. Meskipun **PipeWire** direkomendasikan untuk sistem modern, PulseAudio tetap menjadi pilihan yang stabil untuk kebutuhan audio tradisional.

## 1. Persiapan dan Akses Perangkat

Agar PulseAudio dapat mengakses perangkat keras audio, Anda memiliki dua pilihan:
*   **Grup Audio (Manual):** Tambahkan pengguna Anda ke grup `audio`:
    ```bash
    sudo usermod -aG audio $USER
    ```
*   **Session Manager (Otomatis):** Gunakan pengelola sesi seperti `elogind` yang akan mengatur izin akses secara otomatis saat Anda login.

---

## 2. D-Bus dan Dependensi

Tergantung pada aplikasi yang Anda gunakan, PulseAudio mungkin memerlukan bus sesi D-Bus atau bus sistem D-Bus:
*   **D-Bus Sistem:** Pastikan layanan `dbus` sudah aktif di sistem Anda.
*   **D-Bus Sesi:** Jika lingkungan desktop Anda tidak menyediakannya, Anda mungkin perlu menjalankan sesi dengan `dbus-run-session`.

---

## 3. Kompatibilitas ALSA

Untuk aplikasi lama yang hanya mendukung ALSA secara langsung dan tidak mendukung PulseAudio secara native, Anda dapat menginstal paket berikut:
```bash
sudo xbps-install -S alsa-plugins-pulseaudio
```
Paket ini memungkinkan aplikasi ALSA untuk mengalirkan audio melalui server PulseAudio.

---

## 4. Manajemen Layanan

### **Menjalankan PulseAudio**
Secara default, PulseAudio akan berjalan secara otomatis saat dibutuhkan oleh aplikasi. Jika PulseAudio tidak berjalan otomatis, Anda dapat menjalankannya secara manual dari terminal:
```bash
pulseaudio --daemonize=no --exit-idle-time=-1
```

### **Menonaktifkan Auto-Activation**
Jika Anda tidak ingin PulseAudio aktif secara otomatis (misalnya karena Anda ingin menggunakan PipeWire atau ALSA murni), Anda dapat mengubah pengaturan `autospawn` menjadi `no` pada file konfigurasi klien:

Edit file: `/etc/pulse/client.conf` atau `~/.config/pulse/client.conf`
```text
autospawn = no
```

---

## 5. Alat Bantu (Utility)

Untuk mengelola volume dan perangkat input/output secara grafis, Anda disarankan menginstal:
*   `pavucontrol`: Antarmuka grafis (GTK) untuk mengelola volume.
*   `pulsemixer`: Antarmuka terminal (CLI/Ncurses) yang ringan dan interaktif.
