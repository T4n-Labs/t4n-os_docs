# Graphical Session (Sesi Grafis)

Sesi Grafis adalah lingkungan desktop interaktif yang memungkinkan pengguna berinteraksi dengan sistem operasi melalui antarmuka visual. Dokumen ini mencakup konfigurasi driver grafis, pemilihan display server, hingga pemasangan desktop environment di T4n OS.

## ISI Page

### Driver Grafis
Konfigurasi driver untuk kartu grafis yang digunakan.

- [AMD / ATI](gpu-amd.md)
- [Intel](gpu-intel.md)
- [NVIDIA](gpu-nvidia.md)
- [NVIDIA Optimus](gpu-nvidia-optimus.md)

### Session (Display Server)
Pilih display server yang akan digunakan sebagai fondasi sesi grafis.

- [X11 (Xorg)](xorg.md) — Display server klasik, kompatibilitas luas
- [Wayland](wayland.md) — Display server modern, keamanan lebih baik
- [XDG-Desktop-Portal]() - komponen sesi grafis (session service) 

### GUI Core

- [Base](gui-base.md) — Paket grafis dasar
- [XFCE](xfce.md) — Desktop environment ringan berbasis X11
- [XFCE-Wayland](xfce-wayland.md) — XFCE dengan dukungan Wayland

### GUI Extra
Kustomisasi tampilan sistem.

- [Fonts](fonts.md) — Instalasi dan konfigurasi font
- [Icons](icons.md) — Paket ikon
- [Themes](themes.md) — Tema GTK dan Qt
