# Panduan Kustomisasi Tema - T4n OS

T4n OS memungkinkan fleksibilitas penuh dalam mengubah tampilan antarmuka, mulai dari dekorasi jendela, warna aplikasi, hingga kursor mouse.

## 1. Lokasi Penyimpanan Tema
Sebelum memasang tema, penting untuk mengetahui di mana file tema harus diletakkan:

*   **Seluruh Sistem (System-wide):** `/usr/share/themes/`
    *(Membutuhkan akses root, tersedia untuk semua user)*.
*   **Per Pengguna (User-specific):** `~/.local/share/themes/` atau `~/.themes/`
    *(Hanya tersedia untuk user tersebut)*.

---

## 2. Mengelola Tema GTK (XFCE, GNOME, dll.)
Aplikasi GTK menggunakan file konfigurasi untuk menentukan tema yang aktif.

### **Instalasi melalui XBPS**
T4n OS menyediakan berbagai tema populer di repositori:
```bash
sudo xbps-install -S arc-theme adwaita-icon-theme greybird-themes
```

### **Konfigurasi Manual (Headless/Minimal)**
Jika Anda tidak menggunakan Desktop Environment, edit file `~/.config/gtk-3.0/settings.ini`:
```ini
[Settings]
gtk-theme-name = Arc-Dark
gtk-icon-theme-name = Papirus-Dark
gtk-font-name = DejaVu Sans 11
gtk-cursor-theme-name = Adwaita
```

### **Kustomisasi Grafis (XFCE)**
Pergi ke menu:
1.  **Settings > Appearance:** Pilih tema untuk widget dan ikon.
2.  **Settings > Window Manager:** Pilih tema untuk dekorasi jendela (tombol close, minimize, dll).

---

## 3. Mengelola Tema Qt (KDE, LXQt)
Aplikasi berbasis Qt terkadang terlihat berbeda jika dijalankan di lingkungan GTK. Untuk menyelaraskannya:

1.  **Instal `qt5ct` atau `qt6ct`:**
    ```bash
    sudo xbps-install -S qt5ct qt6ct
    ```
2.  **Atur Variabel Lingkungan:** Tambahkan baris berikut ke `~/.bash_profile` atau `/etc/environment`:
    
```bash
    export QT_QPA_PLATFORMTHEME=qt5ct
    ```
3.  Buka aplikasi **Qt5 Settings** untuk memilih tema yang senada dengan GTK Anda.

---

## 4. Tema Kursor Mouse
Kursor disimpan di direktori `/usr/share/icons/` atau `~/.icons/`.

### **Mengubah Kursor secara Manual**
Buat atau edit file `~/.icons/default/index.theme`:
```ini
[Icon Theme]
Inherits=Nama-Tema-Kursor
```

---

## 5. Tema Terminal
Bagi pengguna T4n OS Base, tampilan terminal dapat diubah melalui variabel lingkungan atau konfigurasi emulator terminal:

*   **XFCE Terminal:** Pergi ke `Edit > Preferences > Colors`.
*   **Akselerasi GPU (Alacritty/Kitty):** Edit file `.yml` atau `.conf` masing-masing di folder `~/.config/`.

---

## 6. Integrasi Mode Gelap (Dark Mode)
Banyak aplikasi modern mendukung skema warna gelap secara sistem. Anda dapat memaksanya melalui variabel lingkungan:

```bash
# Tambahkan ke ~/.profile
export GTK_THEME=Arc-Dark
```

---

## 7. Tips: Mencari Tema Baru
Jika tema di repositori XBPS kurang memuaskan, Anda dapat mengunduh ribuan tema buatan komunitas di:
*   [Gnome-look.org](https://www.gnome-look.org) (untuk GTK)
*   [KDE Store](https://store.kde.org) (untuk Qt/KDE)
*   [XFCE-look.org](https://www.xfce-look.org) (untuk XFCE)

**Cara pasang hasil download:**
1. Ekstrak file `.tar.gz`.
2. Pindahkan folder hasil ekstrak ke `~/.local/share/themes/`.
3. Pilih tema tersebut melalui pengaturan sistem.
