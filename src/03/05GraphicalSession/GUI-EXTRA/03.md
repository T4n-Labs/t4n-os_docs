# Panduan Konfigurasi Ikon - T4n OS

Dokumentasi ini menjelaskan cara mengelola tema ikon untuk aplikasi berbasis GTK pada sistem T4n OS.

## 1. Ikon Bawaan (GTK)
Secara default, aplikasi berbasis GTK akan mencoba menggunakan tema ikon **Adwaita**. 
* Saat Anda menginstal paket `gtk+3`, sistem secara otomatis akan menginstal paket `adwaita-icon-theme` sebagai dependensi standar.

## 2. Menggunakan Tema Ikon Kustom
Jika Anda ingin menggunakan tema ikon lain (seperti *Papirus*, *Arc*, atau *Flat-Remix*), ikuti langkah-langkah berikut:

### **Langkah 1: Instalasi Paket Tema**
Instal tema ikon pilihan Anda melalui manajer paket XBPS:
```bash
sudo xbps-install -S papirus-icon-theme
```

### **Langkah 2: Konfigurasi Sistem**
Anda perlu menentukan nama tema ikon di dalam file konfigurasi GTK. Anda bisa mengaturnya secara global (seluruh sistem) atau spesifik per pengguna:

*   **Per Pengguna (Rekomendasi):** Edit file `~/.config/gtk-3.0/settings.ini`
*   **Seluruh Sistem:** Edit file `/etc/gtk-3.0/settings.ini`

Tambahkan atau ubah baris berikut pada bagian `[Settings]`:
```ini
[Settings]
gtk-icon-theme-name = nama-tema-ikon
```
*(Ganti `nama-tema-ikon` dengan nama folder tema yang terinstal di `/usr/share/icons/`)*.

---

## 3. Menghapus Tema Adwaita
Jika Anda sudah menggunakan tema lain dan ingin menghapus `adwaita-icon-theme`, Anda perlu mengatur sistem agar mengabaikan paket tersebut karena statusnya sebagai dependensi wajib bagi GTK.

1. Tambahkan `adwaita-icon-theme` ke daftar paket yang diabaikan (*ignore*) pada konfigurasi XBPS.
2. Hapus paket tersebut secara manual:
```bash
sudo xbps-remove -F adwaita-icon-theme
```

---

## 4. Referensi Lanjutan
Untuk informasi lebih mendalam mengenai pengaturan properti GTK lainnya, Anda dapat merujuk pada:
*   **Dokumentasi GtkSettings:** Khususnya pada properti `gtk-icon-theme-name`.
*   **Lokasi File:** File ikon biasanya tersimpan di `/usr/share/icons/` (sistem) atau `~/.local/share/icons/` (pengguna).
