# Panduan Konfigurasi Font - T4n OS

Dokumentasi ini menjelaskan cara menginstal, mengelola, dan mengonfigurasi font pada T4n OS agar tampilan teks di sistem grafis Anda optimal.

## 1. Instalasi Font melalui XBPS

T4n OS menyediakan berbagai koleksi font melalui manajer paket XBPS. Berikut adalah beberapa rekomendasi untuk kebutuhan umum:

*   **Standar:** `dejavu-fonts-ttf` atau `xorg-fonts` adalah pilihan dasar yang sangat baik untuk stabilitas sistem.
*   **Multibahasa:** `noto-fonts-ttf` mendukung berbagai bahasa dan skrip di seluruh dunia.
*   **Asia Timur:** `noto-fonts-cjk` menambahkan dukungan untuk bahasa Mandarin (Chinese), Jepang (Japanese), dan Korea (Korean).
*   **Emoji:** `noto-fonts-emoji` untuk dukungan ikon emoji berwarna.
*   **Ikon & Pengembang:** `nerd-fonts` menyediakan font yang sudah dimodifikasi dengan berbagai ikon khusus (cocok untuk terminal atau bar status).

```bash
# Contoh instalasi font dasar dan emoji
sudo xbps-install -S dejavu-fonts-ttf noto-fonts-emoji
```

---

## 2. Instalasi Manual

Jika font yang Anda inginkan tidak tersedia di repositori XBPS, Anda dapat menginstalnya secara manual dengan menyalin file font (`.ttf`, `.otf`) ke direktori berikut:

*   **Sistem (System-wide):** `/usr/share/fonts/` (berlaku untuk semua pengguna).
*   **Pengguna (Per-user):** `~/.local/share/fonts/` (hanya berlaku untuk pengguna saat ini).

Setelah menyalin file font, perbarui cache font sistem dengan perintah:
```bash
fc-cache -fv
```

---

## 3. Kustomisasi Tampilan Font

T4n OS menggunakan **fontconfig** untuk mengatur bagaimana font ditampilkan (seperti *aliasing*, *hinting*, atau mematikan jenis font tertentu).

Konfigurasi yang tersedia berada di: `/usr/share/fontconfig/conf.avail/`.
Untuk mengaktifkannya, Anda harus membuat tautan simbolik (*symlink*) ke direktori: `/etc/fonts/conf.d/`.

### **Contoh: Menonaktifkan Bitmap Fonts**
Bitmap fonts terkadang terlihat pecah pada layar modern. Anda bisa menonaktifkannya (kecuali untuk emoji) dengan langkah berikut:

1.  Buat symlink konfigurasi:
    ```bash
    sudo ln -s /usr/share/fontconfig/conf.avail/70-no-bitmaps-except-emoji.conf /etc/fonts/conf.d/
    ```
2.  Konfigurasi ulang paket `fontconfig`:
    
```bash
    sudo xbps-reconfigure -f fontconfig
```

---

## 4. Verifikasi Konfigurasi

Untuk melihat daftar konfigurasi font apa saja yang saat ini aktif di sistem Anda, gunakan perintah:
```bash
fc-conflist
```

Konfigurasi yang ditandai dengan simbol `+` berarti sedang aktif.
