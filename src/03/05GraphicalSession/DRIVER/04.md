# Panduan Konfigurasi GPU NVIDIA - T4n OS

Di T4n OS, Anda memiliki dua pilihan utama: **Nouveau** (gratis/open-source) untuk stabilitas dan kompatibilitas Wayland, atau **NVIDIA** (proprietary) untuk performa maksimal.

---

## 1. Nouveau (Driver Open-Source)
Driver ini dikembangkan oleh komunitas melalui *reverse engineering*. Sangat direkomendasikan jika Anda menggunakan kartu grafis lama atau ingin menjalankan *compositor* Wayland.

### **Grafis & Render**
*   **OpenGL:** Disediakan oleh paket `mesa-dri`. Paket ini biasanya ada di meta-paket `xorg`, namun wajib diinstal manual jika Anda menggunakan `xorg-minimal` atau Wayland.
*   **Vulkan:** Didukung mulai arsitektur **Kepler (GTX 6xx)** ke atas. Instal `vulkan-loader` dan `mesa-vulkan-nouveau`.
*   **Xorg:** Secara default menggunakan driver `xf86-video-nouveau`. Pada perangkat ARM berbasis Tegra, gunakan driver `modesetting`.

### **Akselerasi Video**
*   **VA-API:** Instal `mesa-vaapi`. Gunakan variabel lingkungan: `LIBVA_DRIVER_NAME=nouveau`.
*   **VDPAU:** Instal `libvdpau-va-gl`. Gunakan variabel lingkungan: `VDPAU_DRIVER=va_gl`.

### **Catatan Reclocking (Performa)**
Nouveau memiliki keterbatasan dalam mengatur kecepatan *clock* (performa):
*   **Mendukung Manual:** Maxwell Gen 1, Kepler, dan beberapa kartu Tesla.
*   **Mendukung Otomatis:** Arsitektur Turing (GTX 16xx) dan yang lebih baru.
*   **Tidak Mendukung:** Maxwell Gen 2 (GTX 9xx) hingga Pascal (GTX 10xx) tidak bisa melakukan *reclocking* karena absennya firmware bertanda tangan digital (*signed firmware*).

---

## 2. NVIDIA (Driver Proprietary)
Direkomendasikan bagi Anda yang membutuhkan performa tinggi (gaming/editing). Driver ini tersedia di repositori **nonfree** dan terintegrasi ke kernel melalui **DKMS**.

### **Identifikasi Driver yang Tepat**
Jalankan perintah berikut di terminal untuk mengetahui model GPU Anda:
```bash
lspci -k -d ::03xx
```

Pilih paket driver berdasarkan keluarga GPU Anda:

| Keluarga GPU | Tipe Driver | Nama Paket |
| :--- | :--- | :--- |
| **Turing (NV160) & terbaru** | Open Source / Proprietary | `nvidia` |
| **Maxwell (NV110) s/d Volta (NV140)** | Proprietary | `nvidia580` |
| **Kepler (NVE0)** | Proprietary (Legacy) | `nvidia470` |
| **Fermi (NVC0)** | Proprietary (Legacy) | `nvidia390` |
| **Tesla (NV50) & lebih lama** | Tidak Didukung | Gunakan **Nouveau** |

---

## 3. Dukungan Aplikasi 32-bit (glibc)
Penting untuk menjalankan game lama atau aplikasi seperti Steam.
*   **Jika menggunakan Nouveau:** Instal `mesa-dri-32bit`.
*   **Jika menggunakan NVIDIA:** Instal `nvidia<versi>-libs-32bit` (contoh: `nvidia470-libs-32bit` atau cukup `nvidia-libs-32bit` untuk versi terbaru).

---

## 4. Berpindah dari NVIDIA ke Nouveau

### **Menghapus Driver NVIDIA**
Instal driver `nouveau` terlebih dahulu, lalu hapus paket NVIDIA yang terinstal:
```bash
sudo xbps-install -S xf86-video-nouveau
sudo xbps-remove nvidia  # Ganti 'nvidia' sesuai versi yang Anda pakai (580, 470, dsb)
```
*Catatan: Jika Anda menghapus driver lawas seperti `nvidia340`, pastikan untuk menginstal kembali paket `libglvnd`.*

### **Menyimpan Kedua Driver (Alternatif)**
Jika Anda ingin tetap menginstal driver NVIDIA tetapi ingin menggunakan Nouveau:
1.  Buka file blacklist (biasanya di `/etc/modprobe.d/nouveau_blacklist.conf` atau `/usr/lib/modprobe.d/nvidia.conf`).
2.  Beri tanda komentar (`#`) pada baris `blacklist nouveau`.
3.  Beritahu Xorg untuk memprioritaskan Nouveau dengan membuat file `/etc/X11/xorg.conf.d/20-nouveau.conf`:

```text
Section "Device"
    Identifier "Nvidia card"
    Driver "nouveau"
EndSection
```

Anda mungkin perlu me-restart sistem Anda agar perubahan ini berlaku.
