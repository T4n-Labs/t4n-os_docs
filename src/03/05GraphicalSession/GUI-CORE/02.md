# Dokumentasi T4n OS XFCE Edition

**T4n OS XFCE** adalah varian unggulan yang menggabungkan efisiensi **T4n OS Base** dengan lingkungan desktop **XFCE4**. Varian ini dirancang untuk pengguna yang menginginkan keseimbangan antara performa tinggi dan kemudahan penggunaan grafis.

## 1. Komponen Utama
Edisi XFCE pada T4n OS menggunakan komponen standar berikut:
*   **Window Manager:** `Xfwm4` (mendukung komposisi grafis).
*   **File Manager:** `Thunar` (ringan dan mendukung kustomisasi plugin).
*   **Panel:** `Xfce4-panel` dengan dukungan applet sistem.
*   **Terminal:** `Xfce4-terminal`.
*   **Display Manager:** `LightDM` (antarmuka login grafis).

---

## 2. Manajemen Sesi Grafis
T4n OS menggunakan **LightDM** untuk mengelola login. 

*   **Memulai Sesi:** Masukkan kredensial Anda pada layar login. XFCE akan dimuat secara otomatis.
*   **Konfigurasi Login:** File konfigurasi utama berada di `/etc/lightdm/lightdm.conf`.
*   **Layanan Sesi:** Pastikan layanan `dbus` dan `lightdm` aktif:
    ```bash
    sudo ln -s /etc/sv/dbus /var/service/
    sudo ln -s /etc/sv/lightdm /var/service/
    ```

---

## 3. Kustomisasi Desktop
XFCE sangat fleksibel. Anda dapat mengatur tampilan melalui **Settings Manager** (`xfce4-settings-manager`):

*   **Appearance (Tampilan):** Mengubah tema GTK dan ikon.
*   **Window Manager:** Mengatur dekorasi jendela dan perilaku *fokus*.
*   **Panel:** Menambah, menghapus, atau mengatur posisi bar panel.
*   **Workspaces:** Mengatur jumlah desktop virtual.

---

## 4. Manajemen Aplikasi & Software
Selain menggunakan `xbps` melalui terminal, Anda dapat mengelola aplikasi dengan cara berikut:

*   **Aplikasi Bawaan:**
    *   Web Browser: Firefox (atau sesuai image build).
    *   Editor Teks: Mousepad.
    *   Archiver: File Roller atau Engrampa.
*   **Menginstal Tool GUI (Opsional):**
    Jika Anda lebih suka mengelola paket secara grafis, Anda dapat menginstal `pamac` (jika tersedia di repo) atau menggunakan `octoxbps`.

---

## 5. Konfigurasi Perangkat Keras di XFCE

### **Display & Monitor**
Gunakan aplikasi **Display** untuk mengatur resolusi dan tata letak multi-monitor. Jika menggunakan laptop, pengaturan kecerahan biasanya dikelola oleh `xfce4-power-manager`.

### **Audio**
T4n OS XFCE menggunakan **Pipewire** atau **PulseAudio**.
*   Gunakan ikon volume di tray (Plugin `xfce4-pulseaudio-plugin`) untuk kontrol cepat.
*   Untuk pengaturan mendalam, jalankan `pavucontrol`.

### **Power Management**
Pengaturan baterai, mode tidur (sleep), dan aksi tutup layar (lid close) dapat ditemukan di **Power Manager**.

---

## 6. Integrasi Thunar (File Manager)
Thunar di T4n OS telah dioptimalkan dengan beberapa fitur:
*   **Custom Actions:** Klik kanan di dalam folder untuk membuka terminal di lokasi tersebut.
*   **Volume Management:** Automount USB drive (memerlukan layanan `gvfs`).
*   **Thumbnail:** Mendukung pratinjau gambar dan video (memerlukan `tumbler`).

---

## 7. Troubleshooting Umum

### **Layar Hitam Setelah Login**
Jika layar menjadi hitam setelah login lewat LightDM:
1. Masuk ke TTY (Ctrl+Alt+F2).
2. Cek apakah partisi penuh: `df -h`.
3. Pastikan driver GPU (Intel/NVIDIA/AMD) sudah terinstal dengan benar sesuai panduan driver T4n OS.

### **Ikon Tray Tidak Muncul**
Pastikan paket `xfce4-statusnotifier-plugin` terinstal dan sudah ditambahkan ke panel Anda.

### **Masalah Izin (Permission) pada Thunar**
Jika Anda tidak bisa mengakses partisi lain atau USB:
Pastikan user Anda berada di grup `storage`:
```bash
sudo usermod -aG storage $USER
```

---

## 8. Perintah Penting Khusus XFCE
| Perintah | Fungsi |
| :--- | :--- |
| `xfce4-panel -r` | Merestart panel jika error atau hang. |
| `xfwm4 --replace &` | Mengganti atau merestart Window Manager. |
| `xfce4-session-logout` | Membuka dialog logout melalui terminal. |
