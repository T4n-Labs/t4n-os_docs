# Manual Pages (man)

T4n OS menggunakan toolset **mandoc** (paket `mdocml`) sebagai pembaca man page default.

---

## Penggunaan Dasar

```bash
man <nama_perintah>
```

Contoh:

```bash
man chroot
```

### Seksi Man Page

Setiap man page termasuk dalam seksi tertentu:

| Seksi | Isi                                          |
|-------|----------------------------------------------|
| 1     | Perintah pengguna (program)                  |
| 2     | System calls                                 |
| 3     | Library calls                                |
| 4     | File khusus (perangkat)                      |
| 5     | Format file dan file konfigurasi             |
| 6     | Game                                         |
| 7     | Ikhtisar, konvensi, dan lain-lain            |
| 8     | Perintah manajemen sistem                    |

Jika ada man page dengan nama yang sama di seksi berbeda, tentukan nomor seksinya:

```bash
man 1 printf   # printf sebagai perintah shell
man 3 printf   # printf sebagai fungsi library C
```

## Mencari Man Page (apropos)

Update database pencarian terlebih dahulu:

```bash
sudo makewhatis
```

Lalu cari man page berdasarkan kata kunci:

```bash
apropos chroot
```

Contoh output:

```
chroot(1)        - run command or interactive shell with special root directory
chroot(2)        - change root directory
xbps-uchroot(1)  - XBPS utility to chroot and bind mount with Linux namespaces
```

> `mdocml` menyertakan cron job `/etc/cron.daily/makewhatis` untuk memperbarui database secara otomatis setiap hari. Pastikan [cron daemon](cron-t4n-os.md) sudah terinstal dan aktif.

---

## Man Page Tambahan

Man page development dan POSIX tidak terinstal secara default. Instal sesuai kebutuhan:

```bash
sudo xbps-install -S man-pages-devel    # Manual pengembangan
sudo xbps-install -S man-pages-posix    # Manual POSIX
```

## Man Page Terlokalisasi

Beberapa paket menyertakan man page dalam bahasa lain, tersedia juga via paket `manpages-*`.

### Konfigurasi Global (semua pengguna)

Tambahkan path ke `/etc/man.conf`. Contoh untuk Bahasa Indonesia:

```
/usr/share/man/id
/usr/share/man/id.UTF-8
```

### Konfigurasi Per Pengguna

Ekspor variabel `MANPATH` di profil shell (`~/.bashrc` atau `~/.zshrc`):

```bash
export MANPATH=/usr/share/man/id:/usr/share/man/id.UTF-8:$MANPATH
```
