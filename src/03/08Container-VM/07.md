# VMware

VMware adalah platform virtualisasi proprietary yang memungkinkan menjalankan virtual machine di atas sistem host. T4n OS dapat digunakan baik sebagai **host** (menjalankan VMware) maupun sebagai **guest** (diinstal di dalam VMware).

---

## T4n OS sebagai Host

### VMware Workstation Player

VMware Workstation Player adalah versi gratis untuk penggunaan personal.

#### Instalasi

Unduh installer dari [vmware.com](https://www.vmware.com/products/workstation-player.html), lalu:

```bash
chmod +x VMware-Player-*.bundle
sudo ./VMware-Player-*.bundle
```

#### Dependensi yang Diperlukan

```bash
sudo xbps-install -S linux-headers gcc make
```

VMware membutuhkan modul kernel yang dikompilasi saat instalasi. Pastikan versi header sesuai dengan kernel yang berjalan:

```bash
uname -r              # Cek versi kernel aktif
xbps-query -Rs linux  # Cek paket kernel tersedia
```

#### Kompilasi Modul Kernel

Jika modul kernel gagal dikompilasi setelah update kernel, jalankan ulang:

```bash
sudo vmware-modconfig --console --install-all
```

#### Aktifkan Layanan VMware

```bash
sudo systemctl start vmware        # Jika menggunakan systemd
# atau via runit:
sudo /etc/init.d/vmware start
```

---

### VMware Workstation Pro

VMware Workstation Pro adalah versi berbayar dengan fitur lebih lengkap (snapshot lanjutan, clone, dsb.). Proses instalasi sama seperti Player.

## T4n OS sebagai Guest (di dalam VMware)

### Instalasi T4n OS di VMware

1. Buat VM baru di VMware
2. Pilih **"Other Linux 5.x or later kernel 64-bit"** sebagai OS type
3. Alokasikan resource:

| Resource | Rekomendasi Minimum | Rekomendasi |
|----------|---------------------|-------------|
| RAM      | 1 GB                | 2 GB+       |
| Storage  | 10 GB               | 20 GB+      |
| CPU      | 1 core              | 2 core+     |

4. Mount ISO T4n OS dan instal seperti biasa

### VMware Tools / open-vm-tools

VMware Tools menyediakan integrasi antara host dan guest:

- Sinkronisasi clipboard
- Drag and drop file
- Resolusi layar otomatis
- Sinkronisasi waktu host-guest
- Performa grafis lebih baik

#### Instalasi

T4n OS menggunakan **open-vm-tools** (versi open-source dari VMware Tools):

```bash
sudo xbps-install -S open-vm-tools
```

Untuk guest dengan sesi grafis (X11/Wayland):

```bash
sudo xbps-install -S open-vm-tools-desktop
```

#### Aktifkan Layanan

```bash
ln -s /etc/sv/vmtoolsd /var/service/
```

Verifikasi open-vm-tools berjalan:

```bash
vmware-toolbox-cmd --version
```

---

### Konfigurasi Resolusi Layar

Jika resolusi layar tidak otomatis menyesuaikan, tambahkan baris berikut ke `/etc/X11/xorg.conf.d/10-vmware.conf`:

```
Section "Device"
    Identifier "VMware SVGA"
    Driver     "vmware"
EndSection
```

Atau gunakan driver `modesetting`:

```
Section "Device"
    Identifier "GPU"
    Driver     "modesetting"
EndSection
```

### Berbagi Folder (Shared Folders)

Aktifkan shared folders di pengaturan VM VMware, lalu mount di guest:

```bash
# Lihat daftar shared folder
vmware-hgfsclient

# Mount shared folder
sudo mkdir -p /mnt/hgfs
sudo vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other,uid=1000
```

Untuk mount otomatis saat boot, tambahkan ke `/etc/fstab`:

```
.host:/    /mnt/hgfs    fuse.vmhgfs-fuse    allow_other,uid=1000    0    0
```

---

### Shared Clipboard & Drag-Drop

Pastikan layanan berikut aktif di dalam guest:

```bash
# Cek status vmtoolsd
sv status vmtoolsd

# Restart jika perlu
sv restart vmtoolsd
```

Jika clipboard masih tidak berfungsi, instal paket tambahan:

```bash
sudo xbps-install -S open-vm-tools-desktop gtkmm
```

## Troubleshooting

### Modul Kernel Gagal Dikompilasi

Pastikan header kernel sesuai dengan versi yang berjalan:

```bash
uname -r
# Contoh output: 6.6.30_1

sudo xbps-install -S linux6.6-headers
sudo vmware-modconfig --console --install-all
```

### Layar Hitam setelah Boot

Coba tambahkan parameter berikut ke kernel command line di `/boot/grub/grub.cfg` atau `/etc/default/grub`:

```
nomodeset
```

Atau paksa penggunaan driver vmware di Xorg:

```bash
sudo xbps-install -S xf86-video-vmware
```

### open-vm-tools Tidak Terdeteksi

Verifikasi modul kernel `vmw_balloon` dan `vmxnet3` sudah dimuat:

```bash
lsmod | grep vmw
modprobe vmw_balloon
modprobe vmxnet3
```

### Waktu Guest Tidak Sinkron

Aktifkan sinkronisasi waktu via VMware Tools:

```bash
vmware-toolbox-cmd timesync enable
```
