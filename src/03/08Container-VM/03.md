# Virtualisasi: libvirt

**libvirt** adalah API dan daemon untuk mengelola platform virtualisasi. Alat ini mendukung berbagai teknologi virtualisasi seperti KVM, QEMU, LXC, Bhyve, Xen, VMWare, hingga Hyper-V.

## 1. Instalasi dan Persiapan

Untuk mulai menggunakan libvirt di T4n OS, ikuti langkah-langkah berikut:

1.  **Instalasi Paket:**
    Instal paket utama libvirt dan D-Bus:
    ```bash
    sudo xbps-install -S libvirt dbus
    ```
    *Catatan: Jika Anda menggunakan QEMU/KVM, pastikan paket `qemu` juga terinstal.*

2.  **Mengaktifkan Layanan (Services):**
    Aktifkan layanan yang diperlukan agar daemon dapat berjalan:
    
```bash
    sudo ln -s /etc/sv/dbus /var/service/
    sudo ln -s /etc/sv/libvirtd /var/service/
    sudo ln -s /etc/sv/virtlockd /var/service/
    sudo ln -s /etc/sv/virtlogd /var/service/
    ```

---

## 2. Manajemen Akses Pengguna

Secara default, sebagian besar perintah manajemen virtualisasi memerlukan hak akses `root`. Namun, Anda dapat menjalankan perintah sebagai pengguna biasa (non-root) dengan cara:

1.  Pastikan paket `polkit` dan `dbus` terinstal serta layanan `dbus` aktif.
2.  Tambahkan pengguna Anda ke grup `libvirt`:
    ```bash
    sudo usermod -aG libvirt $USER
    ```
    *Setelah menjalankan perintah ini, silakan log out dan log in kembali.*

## 3. Antarmuka Manajemen

T4n OS menyediakan dua cara utama untuk mengelola mesin virtual:

### **A. Antarmuka Baris Perintah (CLI) - `virsh`**
Paket `libvirt` menyertakan `virsh`, sebuah shell interaktif dan alat skrip untuk tugas manajemen seperti membuat, mengonfigurasi, dan menjalankan VM, serta mengelola jaringan dan penyimpanan.
*   **Penggunaan Dasar:** `virsh list --all`
*   **Konfigurasi Runtime:** Gunakan `virt-admin` untuk mengonfigurasi daemon `libvirtd` secara langsung.

### **B. Antarmuka Grafis (GUI) - `virt-manager`**
Jika Anda lebih menyukai tampilan visual, instal paket berikut:
```bash
sudo xbps-install -S virt-manager virt-manager-tools
```
`virt-manager` memberikan kemudahan dalam memantau statistik VM serta menyediakan konsol grafis untuk sistem tamu.
