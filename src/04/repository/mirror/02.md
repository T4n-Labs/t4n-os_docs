# Mengganti Mirror di T4n OS

## 📖 Pendahuluan

Setiap repositori memiliki file konfigurasi yang mendefinisikan URL mirror yang digunakan. Untuk repositori resmi Void Linux, file-file ini diinstall oleh package manager di `/usr/share/xbps.d`. Namun, jika ditemukan file duplikat di `/etc/xbps.d`, maka nilai dari `/etc/xbps.d` yang akan digunakan (override).

Ada dua cara untuk mengganti mirror:
1. **Mudah:** Menggunakan `xmirror` (direkomendasikan)
2. **Manual:** Mengedit file konfigurasi secara langsung

---

## 🛠️ Metode 1: Menggunakan `xmirror` (Termudah)

### Instalasi

Jika belum terinstall, pasang paket `xmirror` terlebih dahulu:

```bash
sudo xbps-install -S xmirror
```

### Penggunaan

`xmirror` adalah utilitas yang menangani semua langkah penggantian mirror secara otomatis, termasuk:
- Memperbarui URL di semua file konfigurasi repositori
- Menyinkronkan ulang indeks repositori

**Menjalankan xmirror:**
```bash
sudo xmirror
```

Atau dengan opsi interaktif:
```bash
sudo xmirror -i
```

`xmirror` akan menampilkan daftar mirror yang tersedia dan meminta Anda memilih yang diinginkan. Setelah memilih, semua repositori akan diperbarui secara otomatis.

---

## ✍️ Metode 2: Manual (Alternatif)

Jika ingin melakukannya secara manual, ikuti langkah-langkah berikut:

### Langkah 1: Copy File Konfigurasi ke `/etc/xbps.d`

```bash
sudo mkdir -p /etc/xbps.d
sudo cp /usr/share/xbps.d/*-repository-*.conf /etc/xbps.d/
```

### Langkah 2: Ganti URL Mirror

Edit semua file yang telah di-copy dan ganti URL default dengan URL mirror pilihan Anda.

**Dengan `sed` (otomatis):**
```bash
sudo sed -i 's|https://repo-default.voidlinux.org|https://pilih.mirror.anda.com|g' /etc/xbps.d/*-repository-*.conf
```

**Contoh spesifik:**
```bash
# Mengganti ke mirror di Asia Tenggara (contoh: mirror.labkom.id)
sudo sed -i 's|https://repo-default.voidlinux.org|https://mirror.labkom.id/voidlinux|g' /etc/xbps.d/*-repository-*.conf
```

**Atau edit manual setiap file:**
```bash
sudo nano /etc/xbps.d/00-repository-main.conf
# Ubah baris 'repository=https://repo-default.voidlinux.org/current' 
# menjadi 'repository=https://mirror.pilihan.anda/current'
```

### Langkah 3: Sinkronkan Indeks dengan Mirror Baru

```bash
sudo xbps-install -S
```

Anda akan melihat URL repositori baru muncul selama proses sinkronisasi.

### Langkah 4: Verifikasi Perubahan

Setelah sinkronisasi selesai, verifikasi dengan:

```bash
xbps-query -L
```

**Contoh output:**
```
 9970 https://mirror.pilihan.anda/current (RSA signed)
   27 https://mirror.pilihan.anda/current/multilib/nonfree (RSA signed)
 4230 https://mirror.pilihan.anda/current/multilib (RSA signed)
   47 https://mirror.pilihan.anda/current/nonfree (RSA signed)
 5368 https://mirror.pilihan.anda/current/debug (RSA signed)
```

> Angka di depan menunjukkan jumlah paket di repositori tersebut.

---

## 📂 Struktur File Konfigurasi

| Lokasi | Prioritas | Penggunaan |
|--------|-----------|------------|
| `/usr/share/xbps.d/` | Rendah (default) | Konfigurasi bawaan dari paket, jangan diedit langsung |
| `/etc/xbps.d/` | **Tinggi (override)** | Konfigurasi kustom pengguna, file di sini akan menggantikan yang di `/usr/share/xbps.d` |

> 💡 **Praktik terbaik:** Selalu buat override di `/etc/xbps.d/`, jangan mengedit langsung file di `/usr/share/xbps.d/`. File di `/usr/share/xbps.d/` dapat ditimpa saat update paket.

---

## ⚠️ Catatan Penting

1. **Repositori yang ditambahkan setelahnya** (seperti `nonfree`, `multilib`, `debug`) juga perlu diubah URL-nya, atau mereka akan tetap menggunakan mirror default.

   Jika Anda mengaktifkan repositori tambahan **setelah** mengganti mirror, Anda perlu mengubah URL-nya juga:
   ```bash
   sudo sed -i 's|https://repo-default.voidlinux.org|https://mirror.baru.anda.com|g' /etc/xbps.d/*-repository-*.conf
   ```

2. **Jangan lupa sinkronisasi** setiap kali mengganti mirror (`sudo xbps-install -S`).

3. **Pilih mirror yang terpercaya** - idealnya dari daftar resmi di [xmirror.voidlinux.org](https://xmirror.voidlinux.org).

4. **Kecepatan mirror** - Pilih mirror yang dekat secara geografis untuk kecepatan unduh optimal.

## 🔄 Contoh Lengkap Mengganti Mirror Manual

Berikut contoh lengkap mengganti mirror ke `https://mirror.labkom.id/voidlinux`:

```bash
# 1. Buat direktori konfigurasi jika belum ada
sudo mkdir -p /etc/xbps.d

# 2. Copy file konfigurasi default
sudo cp /usr/share/xbps.d/*-repository-*.conf /etc/xbps.d/

# 3. Ganti URL mirror
sudo sed -i 's|https://repo-default.voidlinux.org|https://mirror.labkom.id/voidlinux|g' /etc/xbps.d/*-repository-*.conf

# 4. Sinkronkan indeks
sudo xbps-install -S

# 5. Verifikasi
xbps-query -L
```

---

## 🧹 Mengembalikan ke Mirror Default

Jika ingin kembali ke mirror default:

```bash
# Hapus file override di /etc/xbps.d
sudo rm /etc/xbps.d/*-repository-*.conf

# Atau pindahkan sebagai backup
sudo mv /etc/xbps.d/*-repository-*.conf /etc/xbps.d/backup/

# Sinkronkan ulang
sudo xbps-install -S
```
