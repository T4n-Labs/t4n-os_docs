# Mirrors (Cermin Server) di T4n OS

## 🌐 Pengertian Mirrors

**Mirrors** adalah server-server cerminan yang menyimpan salinan persis dari repositori paket Void Linux. Mereka tersebar di berbagai wilayah geografis untuk memberikan kecepatan unduh yang lebih baik dan redundansi.

## 📍 Konfigurasi Default

Saat instalasi awal T4n OS, sistem secara default akan menggunakan:
```
repo-default.voidlinux.org
```

Server ini dapat memetakan ke **Tier 1 mirror** mana pun (mirror tingkat pertama yang dikelola langsung oleh proyek Void). Meskipun sudah otomatis, Anda mungkin mendapatkan pengalaman yang lebih baik dengan **memilih mirror yang spesifik dan lebih dekat secara geografis**.

> 💡 **Lihat daftar mirror dan informasi selengkapnya di:**  
> 👉 [xmirror.voidlinux.org](https://xmirror.voidlinux.org)

## 🧅 Tor Mirrors (Repositori via Jaringan Tor)

Void Linux juga menyediakan mirror yang dapat diakses melalui **jaringan Tor** (The Onion Router). Ini berguna untuk:
- Meningkatkan privasi dan anonimitas saat mengunduh paket
- Mengakses repositori di wilayah yang memblokir akses langsung
- Keamanan tambahan di jaringan yang tidak terpercaya

📖 **Panduan lengkap:** [Using Tor Mirrors](https://docs.voidlinux.org/xbps/repositories/mirrors.html#using-tor-mirrors) di dokumentasi resmi Void Linux.

## 🔧 Cara Mengganti Mirror

1. **Lihat mirror yang tersedia** di [xmirror.voidlinux.org](https://xmirror.voidlinux.org)
2. **Edit file konfigurasi** di `/usr/share/xbps.d/00-repository-main.conf` atau buat file override di `/etc/xbps.d/`
3. **Sinkronkan ulang indeks** setelah mengganti:
   ```bash
   sudo xbps-install -S
   ```
