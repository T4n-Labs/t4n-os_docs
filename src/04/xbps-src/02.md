# Cheatsheet xbps-src untuk T4n OS

## 🔧 Perintah Dasar `./xbps-src`

| Perintah | Fungsi |
|----------|--------|
| `./xbps-src binary-bootstrap` | Setup awal environment build |
| `./xbps-src bootstrap-update` | Update bootstrap setelah perubahan |
| `./xbps-src pkg <pkg>` | Bangun paket |
| `./xbps-src clean <pkg>` | Hapus hasil build paket |
| `./xbps-src clean-repocache` | Hapus cache repositori |
| `./xbps-src update-sys` | Update sistem dengan paket hasil build |
| `./xbps-src search` | Cari template yang tersedia |
| `./xbps-src show-options <pkg>` | Tampilkan opsi build yang tersedia |

---

## 📝 Variabel dalam Template (`template` file)

| Variabel | Contoh | Keterangan |
|----------|--------|------------|
| `pkgname` | `firefox` | Nama paket |
| `version` | `120.0_1` | Versi (revisi dipisah underscore) |
| `revision` | `1` | Revisi paket (bukan versi upstream) |
| `wrksrc` | `"${pkgname}-${version}"` | Direktori source setelah ekstrak |
| `build_style` | `gnu-configure`, `cmake`, `python3` | Cara build |
| `configure_args` | `$(vopt_enable alsa)` | Argumen untuk configure script |
| `hostmakedepends` | `pkg-config` | Tools yang dibutuhkan saat build (berjalan di host) |
| `makedepends` | `gtk+3-devel` | Library yang dibutuhkan saat build |
| `depends` | `desktop-file-utils` | Runtime dependencies |
| `short_desc` | `"Web browser"` | Deskripsi singkat (<80 karakter) |
| `maintainer` | `"Your Name <email>"` | Maintainer |
| `license` | `GPL-3.0-or-later` | Lisensi |
| `homepage` | `https://example.com` | URL homepage |
| `distfiles` | `"https://example.com/${pkgname}-${version}.tar.gz"` | Source tarball |
| `checksum` | `sha256...` | SHA256 hash dari distfiles |

---

## 🏗️ Build Styles Umum

| `build_style` | Untuk proyek yang menggunakan... |
|---------------|----------------------------------|
| `gnu-configure` | `./configure` (autotools) |
| `cmake` | CMake |
| `python3` | Setup.py / pyproject.toml |
| `meson` | Meson build system |
| `meta` | Metapackage (tidak build, hanya dependensi) |
| `fetch` | Hanya download file (tanpa build) |

---

## 🔄 Opsi Kustom (vopt)

```bash
# Di template, untuk toggle fitur
vopt_conflict audible extra
vopt_if extra "--enable-extra" "--disable-extra"

# Jalankan build dengan opsi tertentu
./xbps-src -o extra pkg myapp
```

---

## 🧹 Membersihkan

```bash
# Hapus satu paket
./xbps-src clean myapp

# Hapus semua hasil build
rm -rf hostdir/binpkgs/*

# Hapus masterdir (environment build)
rm -rf masterdir
```

---

## 🐛 Debugging

```bash
# Build dengan verbose
./xbps-src -v pkg myapp

# Masuk ke shell dalam chroot build
./xbps-src chroot

# Lihat log build
cat hostdir/logs/myapp-*.log
```

---

## 🔗 Integrasi VUR (Void User Repository)

```bash
# Clone VUR
git clone https://github.com/T4n-Labs/vur
cd vur

# Copy template ke void-packages
cp -r extra/discord ~/void-packages/srcpkgs/

# Build
cd ~/void-packages
./xbps-src pkg discord
```
