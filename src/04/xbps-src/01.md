# xbps-src untuk T4n OS

**xbps-src** adalah sistem build paket untuk XBPS, digunakan untuk membuat paket `.xbps` dari kode sumber (source code). Dengan `xbps-src`, Anda dapat membangun paket sendiri, baik untuk kebutuhan pribadi, testing, maupun kontribusi ke repositori Void Linux.

> 🧅 **T4n OS** berbasis Void Linux, sehingga seluruh dokumentasi ini berlaku penuh.

---

## 🎯 Kapan Anda Perlu xbps-src?

| Situasi | Contoh |
|---------|--------|
| Paket tidak tersedia di repositori | Aplikasi niche atau versi terbaru |
| Perlu opsi build kustom | `--enable-feature` tertentu |
| Ingin paket restricted seperti `virtualbox-ose` | Void hanya sediakan template |
| Mengembangkan paket untuk Void/T4n | Kontribusi template |

---

## ⏱️ Quickstart (5 menit)

### 1. Clone repositori void-packages

```bash
git clone https://github.com/void-linux/void-packages.git
cd void-packages
```

### 2. Bootstrap (siapkan environment build)

```bash
./xbps-src binary-bootstrap
```

### 3. Cari template yang tersedia

```bash
./xbps-src search-pkg firefox
```

### 4. Build paket

```bash
./xbps-src pkg firefox
```

### 5. Install hasil build

```bash
sudo xbps-install --repository=hostdir/binpkgs firefox
```
