# XBPS - Package Manager

X Binary Package System (XBPS) adalah pengelola paket cepat yang telah Dirancang dan diimplementasikan dari awal. XBPS dikelola oleh tim Void Linux.
T4n OS menggunakan **X Binary Package System (XBPS)** sebagai manajer paket utama.

XBPS adalah sistem manajemen paket yang:
- Cepat
- Ringan
- Dibangun dari nol (tidak bergantung pada sistem lain)

Repository resmi:
https://github.com/void-linux/xbps

Sebagian besar pengelolaan paket umum dilakukan dengan perintah-perintah berikut:
- [`xbps-alternatives`](https://man.voidlinux.org/xbps-alternatives)
  Daftar atau menetapkan alternatif yang disediakan oleh paket yang terinstal. Alternatif adalah sistem yang memungkinkan beberapa paket untuk menyediakan fungsionalitas umum melalui Jika tidak, file-file tersebut akan saling bertentangan dengan membuat symlink dari jalur umum ke versi khusus paket yang dipilih oleh pengguna. [Page](./02.md)
- [`xbps-checkvers`](https://man.voidlinux.org/xbps-checkvers) adalah utilitas dalam package manager XBPS yang berfungsi untuk memeriksa paket-paket yang sudah usang (outdated) dengan cara membandingkan versi paket yang ada di repositori atau sistem Anda terhadap versi sumber (source package) yang tersedia di pohon void-packages. [Page](./03.md)
- [`xbps-create`](https://man.voidlinux.org/xbps-create) adalah utilitas dalam package manager XBPS yang berfungsi untuk membuat paket biner XBPS (.xbps) dari file-file yang disimpan dalam suatu direktori. [Page](./04.md)
- [`xbps-dgraph`](https://man.voidlinux.org/xbps-dgraph) adalah utilitas dalam package manager XBPS yang berfungsi untuk menghasilkan grafik dependensi paket dalam format DOT, yang dapat divisualisasikan menggunakan Graphviz. [Page](./05.md)
- [`xbps-digest`](https://man.voidlinux.org/xbps-digest) adalah utilitas dalam package manager XBPS yang berfungsi untuk menghitung message digest (hash) dari sebuah file menggunakan algoritma kriptografi. [Page](./06.md)
- [`xbps-fbulk`](https://man.voidlinux.org/xbps-fbulk) adalah utilitas dalam XBPS yang berfungsi untuk melakukan bulk build (build massal) paket dengan cepat dari repositori void-packages. xbps-fbulk dapat membangun seluruh pohon dependensi dari sebuah paket atau kumpulan paket secara otomatis dan paralel. [Page](./07.md)
- [xbps-fetch](https://man.voidlinux.org/xbps-fetch) adalah utilitas dalam XBPS yang berfungsi untuk mengunduh file dari URL . Ini adalah alat baris perintah sederhana untuk mengambil file dari internet, mirip dengan curl atau wget, tetapi terintegrasi langsung dengan ekosistem XBPS dan menggunakan library libfetch milik NetBSD. [Page](./08.md)
- [`xbps-install`](https://man.voidlinux.org/xbps-install) adalah utilitas utama dalam package manager XBPS (X Binary Package System) untuk menginstal, memperbarui, dan mengelola paket di Void Linux.[Page](./09.md)
- [`xbps-pkgdb`](https://man.voidlinux.org/xbps-pkgdb)
  Dapat melaporkan dan memperbaiki menemukan masalah dalam basis data paket, serta memodifikasinya. [Page](./10.md)
- [`xbps-query`](https://man.voidlinux.org/xbps-query)
  adalah utilitas dalam package manager XBPS yang berfungsi untuk mencari dan menampilkan informasi tentang paket, baik yang sudah terinstall di sistem maupun yang tersedia di repositori remote.[Page](./11.md)
- [`xbps-reconfigure`](https://man.voidlinux.org/xbps-reconfigure)
  adalah utilitas dalam XBPS yang berfungsi untuk menjalankan ulang skrip konfigurasi paket yang sudah terinstall di sistem . Ini adalah alat penting ketika paket mengalami masalah konfigurasi atau ketika Anda perlu menerapkan ulang perubahan konfigurasi .[Page](./12.md)
- [`xbps-remove`](https://man.voidlinux.org/xbps-remove)
  adalah utilitas dalam package manager XBPS yang berfungsi untuk menghapus paket yang sudah terinstall dari sistem Void Linux . [Page](./13.md)
- [`xbps-rindex`](https://man.voidlinux.org/xbps-rindex)
  adalah utilitas dalam XBPS yang berfungsi untuk mengelola repositori paket biner lokal . Dengan kata lain, ini adalah alat yang mengubah folder biasa menjadi repositori XBPS yang dapat dikenali oleh xbps-install. [Page](./14.md)
- [`xbps-uchroot`](https://man.voidlinux.org/xbps-uchroot) adalah utilitas dalam XBPS yang berfungsi untuk melakukan chroot dan bind mount dengan memanfaatkan Linux namespaces untuk isolasi proses. [Page](./15.md)
- [`xbps-uhelper`](https://man.voidlinux.org/xbps-uhelper) adalah utilitas dalam XBPS yang menyediakan berbagai fungsi pembantu (helper) untuk mengolah string dan informasi terkait paket XBPS . Secara sederhana, ini adalah "pisau Swiss army" untuk memanipulasi nama paket, versi, pola dependensi, dan informasi lainnya. [Page](./16.md)
- [`xbps-uunshare`](https://man.voidlinux.org/xbps-uunshare) adalah utilitas dalam XBPS yang berfungsi untuk melakukan chroot dan bind mount ke dalam direktori target dengan memanfaatkan Linux user namespaces . Keunggulan utama dari pendekatan ini adalah pengguna tidak memerlukan hak akses root untuk menjalankannya . [Page](./17.md)

Sebagian besar pertanyaan dapat dijawab dengan merujuk pada halaman manual (man pages) untuk alat-alat tersebut. bersama dengan halaman manual [xbps.d(5)](https://man.voidlinux.org/xbps.d.5).
