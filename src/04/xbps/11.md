## `xbps-query` pada Void Linux

`xbps-query` adalah utilitas dalam package manager XBPS yang berfungsi untuk **mencari dan menampilkan informasi tentang paket**, baik yang sudah terinstall di sistem maupun yang tersedia di repositori remote .

---

## 🎯 Fungsi Utama

`xbps-query` adalah alat **read-only** (hanya membaca, tidak mengubah sistem). Anda menggunakannya untuk:

| Fungsi | Penjelasan |
|--------|------------|
| **Mencari paket** | Mencari paket berdasarkan nama atau deskripsi di repositori |
| **Melihat informasi paket** | Menampilkan detail seperti versi, lisensi, maintainer, dependensi |
| **Melihat daftar paket terinstall** | Menampilkan semua paket yang ada di sistem |
| **Mencari pemilik file** | Menentukan paket mana yang memiliki suatu file |
| **Memeriksa status paket** | Melihat apakah paket di-hold, manual, orphan, atau repolock  |

`xbps-query` sangat penting untuk exploring paket dan memahami keadaan sistem di Void Linux .

---

## 🔧 Mode Operasi (Hanya Satu Mode per Eksekusi)

`xbps-query` beroperasi dalam berbagai mode. Secara default, ia mencari di paket yang **terinstall** di sistem. Jika Anda ingin mencari di repositori remote, tambahkan opsi `-R` .

| Mode | Perintah | Fungsi |
|------|----------|--------|
| **List paket** | `xbps-query -l` | Menampilkan semua paket yang terinstall beserta statusnya (`ii`=terinstall, `uu`=belum dikonfigurasi, `hr`=setengah terhapus)  |
| **List repositori** | `xbps-query -L` | Menampilkan daftar repositori yang aktif dan jumlah paket di dalamnya  |
| **Cari paket** | `xbps-query -Rs <keyword>` | Mencari paket di repositori berdasarkan keyword  |
| **Info paket terinstall** | `xbps-query -S <paket>` | Menampilkan informasi detail paket yang sudah terinstall  |
| **Info paket di repo** | `xbps-query -RS <paket>` | Menampilkan informasi paket dari repositori remote  |
| **File milik paket** | `xbps-query -f <paket>` | Menampilkan daftar file yang diinstall oleh paket  |
| **Dependensi** | `xbps-query -x <paket>` | Menampilkan dependensi yang dibutuhkan paket  |
| **Reverse dependensi** | `xbps-query -X <paket>` | Menampilkan paket apa saja yang membutuhkan paket ini  |
| **Pemilik file** | `xbps-query -o /path/file` | Mencari paket mana yang memiliki file tertentu  |
| **Package orphans** | `xbps-query -O` | Menampilkan paket yang terinstall sebagai dependensi tapi tidak dibutuhkan lagi  |
| **Paket manual** | `xbps-query -m` | Menampilkan paket yang diinstall secara manual oleh pengguna  |
| **Paket di-hold** | `xbps-query -H` | Menampilkan paket yang dikunci agar tidak diupdate  |
| **Paket repolock** | `xbps-query --list-repolock-pkgs` | Menampilkan paket yang terkunci ke repositori tertentu  |

---

## 📋 Contoh Penggunaan

### 1. Mencari Paket di Repositori

```bash
xbps-query -Rs firefox
```

Output akan menunjukkan semua paket yang mengandung kata "firefox" :
- `[*]` artinya paket sudah terinstall
- `[-]` artinya paket belum terinstall

### 2. Mencari Paket dengan Regular Expression

```bash
xbps-query --regex -Rs '^linux[0-9.]+-[0-9._]+'
```
Perintah ini mencari semua kernel yang tersedia, berguna untuk melihat series kernel yang dapat diinstall .

### 3. Melihat Informasi Paket yang Terinstall

```bash
xbps-query -S vim
```

Output akan menampilkan detail seperti versi, deskripsi, arsitektur, maintainer, lisensi, dependensi, dll .

### 4. Melihat Informasi Paket dari Repositori (Belum Terinstall)

```bash
xbps-query -RS firefox
```

### 5. Melihat File-file yang Dimiliki Paket

```bash
xbps-query -f firefox
```
Menampilkan semua file (binary, konfigurasi, documentation) yang diinstall oleh paket Firefox .

### 6. Menentukan Pemilik File

```bash
xbps-query -o /usr/bin/xbps-query
```
Output akan menunjukkan file tersebut dimiliki oleh paket `xbps-0.68_1` .

### 7. Melihat Daftar Repositori Aktif

```bash
xbps-query -L
```

Output akan menampilkan repositori yang dikonfigurasi beserta jumlah paket di dalamnya .

### 8. Melihat Package Orphans (Dependensi Tidak Terpakai)

```bash
xbps-query -O
```

Ini akan menampilkan paket yang terinstall sebagai dependensi tapi tidak lagi dibutuhkan. Paket-paket ini dapat dihapus dengan `xbps-remove -o` .

### 9. Melihat Properti Spesifik Paket

```bash
xbps-query -S firefox -p pkgver,short_desc,license
```
Menampilkan hanya properti yang dipilih (versi, deskripsi singkat, lisensi) .

### 10. Melihat Semua Paket Terinstall

```bash
xbps-query -l
```
Menampilkan daftar lengkap paket dengan status:
- `ii` = terinstall (installed ok)
- `uu` = unpacked (belum dikonfigurasi)
- `hr` = half-removed 

---

## 📦 Package Expression

`xbps-query` mendukung package expression untuk pencarian yang lebih fleksibel :

| Format | Contoh | Penjelasan |
|--------|--------|-------------|
| Nama saja | `firefox` | Versi terbaru |
| Nama + versi eksak | `firefox-120.0_1` | Versi spesifik |
| Lebih besar/sama dengan | `firefox>=120.0` | Versi minimal |

---

## 🗂️ Mode Status Paket

`xbps-query` dapat menampilkan paket berdasarkan mode operasinya :

| Mode | Perintah | Penjelasan |
|------|----------|-------------|
| **Hold** | `xbps-query -H` | Paket dikunci dan tidak akan diupdate dengan `xbps-install -Su` |
| **Manual** | `xbps-query -m` | Paket diinstall manual oleh pengguna (bukan dependency) |
| **Repolock** | `xbps-query --list-repolock-pkgs` | Paket hanya menerima update dari repositori aslinya |

---

## 💡 Perbedaan `-R` vs Tanpa `-R`

| Tanpa `-R` (default) | Dengan `-R` |
|---------------------|-------------|
| Mencari/menampilkan paket yang **sudah terinstall** di sistem | Mencari/menampilkan paket dari **repositori remote** |
| Tidak perlu koneksi internet | Perlu koneksi internet |
| Cepat (baca dari database lokal) | Lambat (fetch dari repo) |

Jadi:
- `xbps-query -S firefox` → info Firefox yang terinstall di sistem anda
- `xbps-query -RS firefox` → info Firefox dari repositori 

---

## 📊 Perbandingan dengan Package Manager Lain

| Fungsi | XBPS (Void) | APT (Debian) | Pacman (Arch) |
|--------|-------------|--------------|---------------|
| Cari paket | `xbps-query -Rs` | `apt search` | `pacman -Ss` |
| Info paket | `xbps-query -RS` | `apt show` | `pacman -Si` |
| Info terinstall | `xbps-query -S` | `apt list` | `pacman -Qi` |
| File milik paket | `xbps-query -f` | `dpkg -L` | `pacman -Ql` |
| Pemilik file | `xbps-query -o` | `dpkg -S` | `pacman -Qo` |
| Dependensi | `xbps-query -x` | `apt depends` | `pacman -Qi` |
| Orphans | `xbps-query -O` | `deborphan` | `pacman -Qdt` |

---

## ⚠️ Catatan Penting

1. **`xbps-query` tidak perlu root** - Perintah ini hanya membaca database, tidak mengubah sistem .

2. **Sinkronisasi repositori diperlukan** - Sebelum menggunakan mode `-R` untuk pencarian, pastikan indeks repositori sudah disinkronkan dengan `xbps-install -S` .

3. **Hanya satu mode per eksekusi** - `xbps-query` tidak bisa menggabungkan beberapa mode flags sekaligus .

4. **Case-sensitive** - Pencarian nama paket bersifat case-sensitive.

---

## 🔗 Kesimpulan

`xbps-query` adalah alat **pencarian dan informasi** yang esensial di Void Linux. Anda akan menggunakannya setiap kali ingin:
- Mencari paket untuk diinstall (`xbps-query -Rs` )
- Memeriksa detail paket sebelum install (`xbps-query -RS` )
- Melihat file apa saja yang dimiliki paket (`xbps-query -f`)
- Mencari tahu paket mana yang memiliki suatu file (`xbps-query -o`)
- Mengecek package orphans untuk dibersihkan (`xbps-query -O` )

`xbps-query` adalah jendela Anda untuk melihat isi repositori dan sistem paket di Void Linux.
