## `xbps-checkvers` pada Void Linux

`xbps-checkvers` adalah utilitas dalam package manager XBPS yang berfungsi untuk **memeriksa paket-paket yang sudah usang (outdated)** dengan cara membandingkan versi paket yang ada di repositori atau sistem Anda terhadap versi sumber (source package) yang tersedia di pohon `void-packages`.

---

## 🎯 Apa yang Dilakukan `xbps-checkvers`?

Utilitas ini membandingkan dua sumber versi:

| Sumber | Deskripsi |
|--------|-----------|
| **Versi dari repositori/pkgdb** | Versi paket yang tersedia di repositori remote (atau yang sudah terinstall di sistem) |
| **Versi dari void-packages tree** | Versi source package terbaru dari koleksi template build Void Linux |

Dengan perbandingan ini, Anda bisa mengetahui paket mana yang sudah ketinggalan versi dibandingkan dengan source upstream.

---

## 🔧 Parameter Penting

| Parameter | Fungsi |
|-----------|--------|
| `-s, --show-all` | Menampilkan SEMUA paket dari `void-packages` tree beserta versi di repositori dan versi sumbernya |
| `-I, --installed` | Membandingkan paket yang **sudah terinstall** (bukan yang di repositori) terhadap source tree |
| `-D, --distdir <dir>` | Menentukan lokasi `void-packages` tree (default: `~/void-packages`) |
| `-f, --format <format>` | Mengatur format output (mirip `printf`) dengan kode seperti `%n` (nama pkg), `%r` (versi repo), `%s` (versi source) |
| `-e, --removed` | Menampilkan paket yang ada di repositori tapi sudah dihapus dari `void-packages` tree |
| `-m, --manual` | Hanya memproses paket-paket yang disebutkan secara manual |
| `--staging` | Menggunakan staged packages dari repositori remote |

---

## 📋 Contoh Penggunaan

### 1. Melihat semua paket dengan versi dari repo dan source
```bash
xbps-checkvers -s
```
Output akan menampilkan tiga kolom: `nama_paket versi_repo versi_source`

### 2. Cek paket yang sudah terinstall apakah ada versi lebih baru di source tree
```bash
xbps-checkvers -I
```

### 3. Cek paket tertentu saja
```bash
xbps-checkvers -I -m firefox
```

### 4. Menggunakan void-packages tree di lokasi khusus
```bash
xbps-checkvers -D ~/src/void-packages -I
```

### 5. Format output kustom
```bash
xbps-checkvers -f "%n -> repo: %r, source: %s\n"
```

## 💡 Siapa yang Membutuhkan Tools Ini?

`xbps-checkvers` sangat berguna untuk:

| Pengguna | Kegunaan |
|----------|----------|
| **Maintainer paket Void Linux** | Mengecek paket mana yang perlu di-update di repositori |
| **Pengguna yang mem-build dari source** | Membandingkan versi terinstall dengan versi source terbaru |
| **Contributor void-packages** | Memastikan template build sudah up-to-date |

---

## 📂 Kebutuhan Sebelum Penggunaan

Agar `xbps-checkvers` berfungsi optimal, Anda memerlukan **`void-packages` repository** (biasanya di `~/void-packages`). Ini adalah koleksi template build yang digunakan Void Linux untuk membangun semua paketnya.

Jika tidak ada, Anda bisa meng-clone-nya:
```bash
git clone https://github.com/void-linux/void-packages.git ~/void-packages
```

---

## ⚠️ Catatan Penting

1. **Secara default**, `xbps-checkvers` membandingkan paket dari **repositori** (bukan yang sudah terinstall) dengan source tree
2. Gunakan `-I` jika ingin membandingkan **paket yang sudah terinstall** di sistem Anda
3. Output `?` menandakan versi tidak dapat ditemukan (mungkin paket sudah dihapus dari repositori)
4. Tools ini adalah bagian dari keluarga perintah XBPS, bersama dengan `xbps-install`, `xbps-query`, `xbps-remove`, dll

---

**Kesimpulan**: `xbps-checkvers` adalah alat untuk maintainer dan power user Void Linux yang ingin memantau ketersediaan versi terbaru paket dengan membandingkan repositori/instalasi lokal terhadap source template build upstream.
