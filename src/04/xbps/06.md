## `xbps-digest` pada Void Linux

**`xbps-digest`** adalah utilitas dalam package manager XBPS yang berfungsi untuk **menghitung message digest (hash) dari sebuah file** menggunakan algoritma kriptografi .

---

## 🎯 Fungsi Utama

Utilitas ini pada dasarnya adalah alat untuk menghasilkan **checksum/hash** dari file tertentu. Ini sangat berguna untuk:

| Kegunaan | Penjelasan |
|----------|------------|
| **Verifikasi integritas file** | Memastikan file tidak berubah/corrupt |
| **Membandingkan file** | Dua file yang identik akan menghasilkan hash yang sama |
| **Validasi download** | Mengecek apakah file yang diunduh sesuai dengan aslinya |

---

## 🔧 Parameter yang Tersedia

### Mode Hash

| Parameter | Fungsi |
|-----------|--------|
| `-m, --mode mode` | Menentukan algoritma hash. **Saat ini hanya mendukung `sha256`** . Jika tidak ditentukan, defaultnya adalah `sha256` |

### Opsi Lainnya

| Parameter | Fungsi |
|-----------|--------|
| `-h, --help` | Menampilkan pesan bantuan |
| `-V, --version` | Menampilkan informasi versi |

---

## 📋 Cara Penggunaan

Sintaks dasar:
```bash
xbps-digest [OPTIONS] [FILE] [FILE+N]
```

Jika tidak ada file yang ditentukan, `xbps-digest` akan membaca dari **stdin** (input standar) .

### Contoh Penggunaan

**1. Menghitung hash dari satu file**
```bash
xbps-digest dokumen.pdf
```
Output: hash SHA-256 dari file `dokumen.pdf`

**2. Menghitung hash dari beberapa file sekaligus**
```bash
xbps-digest file1.txt file2.txt file3.txt
```
Output: hash SHA-256 untuk setiap file (dipisah baris)

**3. Menentukan mode hash secara eksplisit**
```bash
xbps-digest -m sha256 image.iso
```

**4. Membaca dari stdin**
```bash
echo "Hello World" | xbps-digest
```

**5. Verifikasi file yang didownload**
```bash
# Hitung hash file yang baru diunduh
xbps-digest program.xbps

# Bandingkan dengan hash yang diketahui (misal dari website)
echo "a1b2c3... program.xbps" | sha256sum -c
```

---

## 💡 Perbedaan dengan `xbps-uhelper digest`

`xbps-digest` adalah utilitas **pengganti** dari `xbps-uhelper digest` . Jika Anda menggunakan Void Linux versi lama, mungkin masih menemukan `xbps-uhelper digest`. Namun, disarankan untuk beralih ke `xbps-digest` karena ini adalah versi terbaru dan terstandarisasi.

## ⚠️ Catatan Penting

1. **Hanya mendukung SHA-256** - Tidak seperti beberapa tools hash lain yang menyediakan md5, sha1, dll, `xbps-digest` saat ini hanya mengimplementasikan SHA-256 .

2. **Tidak perlu root** - Perintah ini dapat dijalankan sebagai user biasa.

3. **File tidak perlu terkait XBPS** - Anda bisa menghitung hash file apapun, tidak hanya paket `.xbps`.

---

## 🔗 Kesimpulan

`xbps-digest` adalah utilitas sederhana namun berguna dalam ekosistem XBPS, yang memungkinkan pengguna untuk:
- Menghitung hash SHA-256 dari file apapun
- Dapat diintegrasikan dalam script untuk verifikasi integritas
- Membantu dalam proses troubleshooting paket yang corrupt

Meskipun fungsinya terbatas pada satu algoritma hash, kebutuhan utama dalam ekosistem paket modern (termasuk XBPS) memang **SHA-256** sebagai standar keamanan yang kuat.
